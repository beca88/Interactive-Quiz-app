# Interactive Quiz Application

A full-stack interactive quiz application featuring a dynamic web user interface built with **React (Vite)** and a robust database-driven backend API engineered using **ASP.NET Core Web API** and **Entity Framework Core**.

## 🚀 Features

- **Dynamic Quiz Gameplay:** Features slick multiple-choice question rendering with instant evaluation tracking.
- **Interactive State UI:** Styled components providing real-time visual feedback (green highlights for correct answers, red for incorrect selections) with automatic navigation mechanics.
- **Automated Score Tracker:** Accurately evaluates individual responses asynchronously to compute and summarize final score breakdowns.
- **Relational Database Management:** Powered by SQL Server with programmatic seed data population using Entity Framework Core Migrations.
- **Cross-Origin Resource Sharing (CORS):** Fully integrated middleware security configurations allowing communication between isolated local runtime ports.

---

## 🛠️ Tech Stack

### Frontend
- **Framework:** React.js (via Vite)
- **Styling:** Custom CSS3 (Flexbox-driven layout architecture with root variable configurations)
- **Data Fetching:** Native JavaScript Fetch API

### Backend
- **Framework:** ASP.NET Core Web API (.NET 10.0)
- **ORM:** Entity Framework Core
- **Database:** Microsoft SQL Server (LocalDB)
- **Documentation:** Swagger UI (Swashbuckle)

## 🔧 Quick Start

### Backend API
1. Navigate to the backend directory: `cd backend`
2. Update the database schema: `dotnet ef database update`
3. Run the server: `dotnet run` 
*(The API will launch locally; check your terminal output for the active local URL).*

### Frontend UI
1. Navigate to the frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Launch the app: `npm run dev` 
*(Open the local server link generated by Vite in your browser).*
